# Development Guide

## Release Checklist

- Run `just check`.
- Run the release task: `just release v<major.minor.path>`. Such `just release v0.1.7`.
- Check if [Continuous Integration][ci] workflow is completed successfully.
- Push the tags: `git push --tags`
